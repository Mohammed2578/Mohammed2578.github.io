what is one way you can identify a div

You can identify a div in HTML by giving it a unique `id` attribute, like `div id=example`, which allows you to target it specifically with CSS or JavaScript.

why would you use a div element

You use a div element to group and organize other HTML elements together, helping to structure the webpage and apply styles or scripts to that section as a whole. It’s a versatile container used for layout and styling purposes.

what is border property

The **border** property in CSS is used to set the style, width, and color of the border around an HTML element. For example, border: 2px solid black creates a solid black border that’s 2 pixels thick.

what are some of the different border properties

The **border-width** property sets how thick the border is. The **border-style** defines the type of border, like solid or dashed. The **border-color** sets the border’s color. The **border-radius** rounds the corners of the border. You can also set borders on specific sides using **border-top**, **border-right**, **border-bottom**, or **border-left**.

what is the difference between margins and padding

Margins create space **outside** an element’s border, pushing other elements away, while padding creates space **inside** the element’s border, pushing the content away from the edges.
